# ManagementSystem---RecRoom
This is a Waitlist management system designed to automate the process of waitlisting at the Student Union Recreational Center. It is basically used for Billiards and Bowling.

# Problem Statement
To avoid frequent customer inquiries, interruption of music and unresponsive calls

# Solution 
Web-based Waitlist Management System 
1. Customers register for their respective games(billiards,  Bowling).
2. A token is generated for tracking the queue number of the customer.
3. The notification is sent to the customer regarding their turn.
4. Admin login for staff to monitor and manage the bookings.

# Tech Stack 
Front end: React
Backend: NodeJS, SQL
